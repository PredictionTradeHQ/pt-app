## 🔧 Características Técnicas

### Cliente HTTP (`lib/pms.ts`)
- Transformación automática de datos PMS → UI format
- Caché inteligente (revalidación 60s)
- Manejo robusto de errores
- Clasificación automática de categorías con análisis de keywords

### WebSocket (`hooks/use-pms-websocket.ts`)
- Conexión automática
- Reconexión con exponential backoff (max 5 intentos)
- Estados: connecting, connected, error, disconnected
- Callbacks para precios, transacciones, y book de órdenes

### Endpoints (`app/api/pms/*`)
- `/api/pms` - Listar mercados con filtros
- `/api/pms?id=X` - Obtener mercado específico
- `/api/pms/X/history` - Historial de precios
- `/api/pms/X/odds` - Odds en tiempo real
