# PT Tone Guide — Internal Guardrail

This is the single source of truth for how Prediction Trade sounds.
All skills, prompts, AI outputs, in-app copy, and automation must pass this filter.

---

## What PT is

A social platform where people build a reputation for being right about the future.
Mental model: Chess.com meets Twitter meets Duolingo.

Users are **forecasters**. Forecasting is a skill, a habit, and a social identity.
PT is where that identity lives, grows, and gets shared.

## What PT is NOT

- A trading terminal
- An investment or brokerage platform
- A crypto exchange
- A financial research tool
- A paper trading simulator (even though virtual money exists)
- An enterprise SaaS product

---

## Voice

| Dimension | Description |
|---|---|
| Tone | Confident, energetic, human. Smart friend, not financial advisor. |
| Pacing | Short sentences. Punchy. Direct. |
| Personality | Celebratory of skill. Competitive but playful. Data-informed but never cold. |
| Point of view | Always on the forecaster's side — their identity, their wins, their reputation. |

---

## PT sounds like this ✓

- "You called it. 7 days in a row."
- "Predict the future. Build your reputation."
- "Only 4% of forecasters hit this streak."
- "Your accuracy: 73%. Top 12% globally."
- "Will it happen? You decide."
- "One more prediction keeps your streak alive."
- "You've been right more often than most."
- "Called it. Now prove you can do it again."
- "Your record speaks for itself."

## PT does NOT sound like this ✗

- "Your position has been settled at market close."
- "Portfolio P&L: +$1,240 (unrealized)"
- "Market participants are pricing in a 65% probability."
- "Trading volume indicates increased liquidity in the YES contract."
- "Leverage your alpha-generating insights to optimize your forecasting portfolio."
- "Risk-adjusted returns suggest a contrarian stance."
- "Please note: this platform does not constitute financial advice."
- "Settlement occurs upon market resolution."

## Vocabulary

### Allowed ✓

| Use this | Because |
|---|---|
| Forecaster | Social identity, skill-based |
| Prediction | Clear, neutral, accessible |
| Called It | Celebratory, iconic, shareable — PT's signature moment |
| Streak | Habit, daily engagement loop |
| Accuracy | Skill metric, not financial |
| Reputation | Social capital |
| Leaderboard | Competition, fun |
| Badge | Achievement, identity |
| Community | Social, belonging |
| Market | Acceptable — refers to topics/questions, not instruments |
| Outcome | What happens — neutral |
| Record | Historical performance |
| Resolve / Resolution | What happens to a market — acceptable, non-financial |

### Prohibited ✗

| Never use | Because |
|---|---|
| Position | Trading terminal language |
| Portfolio | Investment framing |
| P&L / Profit & Loss | Financial instrument framing |
| Trade / Trading | Implies financial transaction |
| Liquidity | B2B / market structure jargon |
| Alpha | Finance language |
| Settlement | Legal/financial process language |
| Contract | Legal/financial framing |
| Leverage | Financial product term |
| Risk-adjusted | Compliance/investment language |
| Bull / Bear | Trading community signaling |
| Long / Short | Position framing |
| Disclaimer / Not financial advice | Creates broker association |
| Paper trading | Reduces perceived value of the social layer |

## Framing test

Every piece of copy must pass this single test:

> **"Does this make the user feel like a skilled person in a community, or like a trader in a market?"**

### Social framing ✓
- Centers the user's identity and reputation
- Celebrates accuracy as a learnable skill
- Uses community benchmarks ("top 15%", "more than most forecasters...")
- Creates FOMO around streaks and consistency
- Invites sharing — bragging rights are the point

### Financial framing ✗
- Centers money, positions, or portfolio value
- References market mechanics or contract settlement
- Uses risk/return language
- Implies monetary decisions or financial consequences

## AI output rules

When any AI (Claude, workflow, skill, automation) generates content for PT:

1. No financial disclaimers unless legally required by an external party
2. No references to "trading" as a primary activity
3. No market structure language (liquidity, volume, spread, settlement)
4. Always use "forecaster" — never "trader" — for PT users
5. Always use "prediction" — never "trade" or "position" — for user actions
6. Celebrate accuracy and streaks, never profit or loss
7. Keep in-app copy under 3 sentences
8. Sanity check: would this sentence sound natural on a sports prediction app? If yes → OK. Would it sound natural on a brokerage website? If yes → rewrite completely.
