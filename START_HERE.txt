🚀 PRIMEROS PASOS (5 MINUTOS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CONFIGURAR ENTORNO
   $ cp .env.local.example .env.local
   
   Luego edita .env.local e ingresa tu API key:
   https://www.predictionmarkets.market/docs

2. INICIAR SERVIDOR
   $ pnpm dev
   
   El servidor correrá en http://localhost:3000

3. VERIFICAR FUNCIONAMIENTO
   • Abre http://localhost:3000/predict
   • Abre DevTools (F12) → Console
   • Deberías ver mercados cargando y logs de WebSocket

→ Historial de precios del mercado

$ curl "http://localhost:3000/api/pms/market_id/odds"
→ Odds actuales del mercado
