🔌 CONFIGURACIÓN DE ENTORNO

Archivo: .env.local.example (creado)

Variables requeridas:
  • PMS_API_KEY = Tu API Key de https://www.predictionmarkets.market/docs
  • PMS_API_URL = https://api.predictionmarkets.market/v1
  • PMS_WS_URL = wss://ws.predictionmarkets.market/v1/stream

Instrucciones:
  1. cp .env.local.example .env.local
  2. Edita .env.local y agrega PMS_API_KEY
  3. Guarda el archivo
  4. pnpm dev

No hay categorías                → Recarga la página
